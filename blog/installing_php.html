Installing PHP in Linux

  1. Login into your server with your credentials.
  2. Type “sudo apt install php libapache2-mod-php php-mysql -y”. This will install an Apache module that allows PHP code to be executed within an Apache web server and a PHP module that allows PHP to communicate with MySQL databases.
  3. To confirm that PHP was installed we can do it with the version command: “php -v”.
