Installing Apache in Linux

  1. Having previously installed Ubuntu Server in your PC, log in with your user and password.
  2. After login in, type the following command to update our package repository in this machine: “sudo apt update”.
    We will need to provide the password required
  3. Once done, we need to install Apache. So, we will type the following command to do it: “sudo apt install apache2”.
    When a prompt appears, we type “Y”.
  4. To verify if it is working, in our browser we type “localhost” and the default page from Apache should appear:
  5. We can also check the status of apache2 service with the following command: “sudo systemctl status apache2”
