Configuring a SSL self-signed certificate in Linux using Apache

  1. Login into your server with your credentials.
  2. We need to install the SSL module for Apache. The command is the following: “sudo apt install apache2 ssl-cert”
  3. Then, we need to enable the SSL module for Apache. The command is: “sudo a2enmod ssl”.
  4. It tells us to restart apache2. We will do it.
  5. Now, we have to create the SSL certificate. This is the commad:
    sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout /etc/ssl/private/apache-selfsigned.key -out /etc/ssl/certs/apache-selfsigned.crt
    This will show up. The most important part is the Common Name part, where we need to specify the site's host
  6. We need to edit the firstpage.info.conf file. Use this command:
    "sudo nano /etc/apache2/sites-available/your_domain_or_ip.conf"
  7. Now, we have to reload apache
    sudo systemctl reload apache2
